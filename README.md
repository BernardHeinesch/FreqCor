# FREQCOR

FREQCOR (**FREQ**uency **COR**rection) is a Python toolbox to compute and apply spectral/cospectral correction factors to eddy covariance fluxes.

It produces:

- cut-off frequency look-up tables (LUTs), and
- correction factor LUTs (stable/unstable), which can then be applied to half-hourly flux time series.

Two types of inputs are currently supported: (i) outputs from an EddyPro run (binned (co)spectra and the full_output file), or (ii) outputs from the GEddySoft software (https://github.com/BernardHeinesch/GEddySoft). Handling of other input formats can be easily implemented by replacing the read routine by your own.

## Installation

### Option 1: pip + venv

From the repository root:

```bash
python -m venv .venv
# Windows PowerShell:
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### Option 2: conda

```bash
conda create -n freqcor python=3.10
conda activate freqcor
pip install -r requirements.txt
```

## Quick start

1. Use the example configuration:

   `examples/metadata/FREQCOR_config_example.ini`

2. Run the entry point:

   - Open `src/FREQCOR_Start.py`.
   - In the *MANUAL MODE* section, set `ini_file` to the example INI path.
   - Run `FREQCOR_Start.py`.

Outputs are written under `examples/output/` (or under the `output_path` configured in the INI file).

## Example dataset

The repository ships with an example dataset from the ICOS Lonzée station (BE-Lon). It includes EddyPro-generated
inputs covering two spring months of 2024 over a winter wheat crop.

The example INI file is configured for a **CO2 cospectral** run on this dataset.
